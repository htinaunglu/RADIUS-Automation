# RADIUS-Automation
### INFO: 
Automation script to update the expiration dates for a batch of users
### SUPPORTED SOFTWARE AND VERSION:
Script only available for "DMA Softlab's Radius Manager 4".

- [rfile.py](rfile.py)

- [users.txt](users.txt)
- - Text file where you put your list of users.
